# Forum Pageviews Visualizer

Visualizes daily page views of the freeCodeCamp.org forum from May 2016 to December 2019 using Python.

## Features

- **Line Plot**: Shows daily page views over time.
- **Bar Plot**: Shows average monthly page views grouped by year.
- **Box Plots**: Show yearly trends and monthly seasonality.

## Technologies

- Python
- Pandas
- Matplotlib
- Seaborn

## Usage

1. Make sure `fcc-forum-pageviews.csv` is in the project folder.
2. Install required packages:

pip install pandas matplotlib seaborn

Run the main script:

python main.py

Plots will be saved as line_plot.png, bar_plot.png, and box_plot.png.
